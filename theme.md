# Design System: Violet Habit Gamification

## Core Concept
A high-energy, rewarding experience using a "Violet" core palette to evoke growth, spirituality, and focus.

## Color Palette
- **Primary (Violet):** `#7C3AED` (Active States)
- **Secondary (Lavender):** `#C4B5FD` (Secondary Accents)
- **Background (Deep Space):** `#0F172A`
- **Surface (Card):** `#1E293B`
- **Text (Primary):** `#F8FAFC`
- **Text (Muted):** `#94A3B8`
- **Success (XP Green):** `#22C55E`
- **Alert (Warning):** `#F59E0B`

## Typography
- **Headings:** Inter (Bold/ExtraBold)
- **Body:** Inter (Regular/Medium)
- **Labels/UI Counters:** DM Mono (Monospace for numeric values/levels)

## Spacing (Tailwind Scale)
- **Base:** 4px units (e.g., `p-4` = 16px)
- **Cards:** `rounded-2xl` border radius.

## Component Guidelines
- **Elevated Rows:** Use `bg-slate-800` with `shadow-lg` and subtle border `border-slate-700`.
- **Status Badges:** Use `text-xs font-mono uppercase tracking-wider` with `bg-violet-900/50 text-violet-200`.
- **Progress Bars:** Base height `h-2`, background `bg-slate-700`, fill `bg-gradient-to-r from-violet-500 to-indigo-400`.
